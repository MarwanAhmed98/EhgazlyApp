# 🚀 Ehgazly App

A modern web application designed to simplify court booking and management, providing users with a seamless and intuitive experience for reserving sports courts.

---

## 📌 Overview

Ehgazly App is a frontend-focused application built to streamline the process of browsing, selecting, and booking courts. The project emphasizes clean UI, dynamic interactions, and scalable architecture using Angular.

---

## ✨ Features

* 🏟️ Browse available courts with dynamic data rendering
* 📅 Select dates and manage bookings
* 🔄 Real-time UI updates based on user interaction
* 🧠 Smart state handling for selected courts
* ✅ Form validation and user-friendly feedback
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Framework:** Angular 19
* **Languages:** TypeScript, JavaScript
* **Styling:** Tailwind CSS
* **Tools:** Git, GitHub

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/MarwanAhmed98/EhgazlyApp.git

# Navigate into project
cd EhgazlyApp

# Install dependencies
npm install

# Run the project
ng serve
```

---

## 🚀 Usage

1. Open the application in your browser
2. Browse available courts
3. Select your preferred court
4. Choose date and interact with booking UI
